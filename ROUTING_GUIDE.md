# 라우팅 가이드

## 현재 회원 전용 메뉴

다음 3개의 메뉴가 회원 전용으로 설정되어 있습니다:

1. **대시보드** (`/dashboard`)
2. **회원권 관리** (`/membership`)
3. **프로젝트 문서** (`/docs`)

## 동작 방식

### 로그인하지 않은 사용자
- 회원 전용 메뉴 클릭 시 → 로그인 페이지로 리다이렉트
- URL에 `returnTo` 파라미터 포함 (예: `/?returnTo=/dashboard`)
- 로그인 성공 후 원래 가려던 페이지로 자동 이동

### 로그인한 사용자
- 회원 전용 메뉴 클릭 시 → 해당 페이지로 이동
- 자동으로 토큰 갱신 확인

## 문제 해결

### 메뉴가 반응하지 않는 경우

1. **브라우저 콘솔 확인**
   - F12를 눌러 개발자 도구 열기
   - Console 탭에서 에러 메시지 확인

2. **로그인 상태 확인**
   - localStorage에 토큰이 있는지 확인
   - `ilc_access_token`과 `ilc_refresh_token` 키 확인

3. **페이지 새로고침**
   - Ctrl + F5로 강력 새로고침

4. **서버 실행 확인**
   - 백엔드 서버가 실행 중인지 확인
   - 프론트엔드 서버가 실행 중인지 확인
